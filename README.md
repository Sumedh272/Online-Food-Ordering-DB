# Online Food Ordering System Database

A relational database schema built in MySQL to model a food delivery application. 

## Schema Components
* **Core Entities:** Customers, Restaurants, MenuItems, Orders, OrderDetails, Reviews.
* **Key Features:** Primary/Foreign key relationships, data type optimization, and complex multi-table JOIN queries.

## File Included
* `online_food_ordering_system.sql`: Standalone script containing table definitions, sample records, and analytical queries.
